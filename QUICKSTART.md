# Quick Start Guide

## Using Study Tracker RIGHT NOW (localStorage mode)

### Step 1: Open the App

Just double-click on `study-tracker.html` or run:

```bash
open study-tracker.html
```

That's it! The app is ready to use.

### Step 2: Create an Account

1. Click on the "Account" tab
2. Click "Sign Up"
3. Enter your name, email, and password
4. Click "Create Account"

### Step 3: Start Tracking

You can now:
- Use the Pomodoro timer
- Log study sessions
- Create weekly plans
- View your study statistics

All data is saved automatically in your browser.

## When You're Ready for Coolify Deployment

See `DEPLOYMENT.md` for full instructions.

Quick version:

1. **Install Node.js** (if needed)
2. **Push to GitHub**:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin YOUR_REPO_URL
   git push -u origin main
   ```
3. **Deploy in Coolify**:
   - Add new application
   - Point to GitHub repo
   - Set environment variables
   - Deploy!
